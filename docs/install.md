## Установка зависимостей

Фронденд сайта собирает `gulp`, его имеет смысл ставить глобально.

```bash
$ sudo npm i -g gulp
```

Для установки зависимостей достаточно выполнить в корне проекта

```bash
$ npm i
```

## Сборка фронтенда

```bash
$ gulp
```
